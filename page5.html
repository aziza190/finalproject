The Process

What was the Assignment?

Create a new Github repository. I prepared three lines about the food, including information about the origins of ethnic cuisine, how it is produced, and how to prepare various dishes.

What was the most fun part of the project?

The ability to code something I enjoy eating as well as learning how to build the repository with that name and adding the boiler template for the first time was the most enjoyable aspect of the project.

What was the toughest part of the project?

Setting up the boiler template proved to be the most challenging because I was just learning how to code and how to properly nest inside of both the div section and the boiler template.
