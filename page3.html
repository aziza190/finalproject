The Process

What was the Assignment?

The task was to create a website that was an identical clone of https://www.xbox.com/en-US/accessories's website, complete with rows and columns for product categories, carousels for the navigation bar, and a completely functional website.

What was the most fun part of the project?

Building this website piece by section and learning how to design a website using a basic template and an HTML document, as well as adding columns and rows, was the most enjoyable aspect of the process.

What was the toughest part of the project?

Making the rows, dividing them into divisions, then adding the div row and three divs for the items proved to be really challenging for me. . In the end, I sought assistance from the resources bootstrap and w3 schools for adding the product name, price, and learn more link.

Characteristics

Characteristics 1

When working individually on this XBOX task, independence is a critical component of getting the assignment completed.

Characteristics 2

a desire to get better learning to ask for assistance from the instructor and getting better at front end coding are all aspects of my ongoing improvement

Characteristics 3

Taking the Lead, Adopting a Growth Mindset, and Being Consistent When I Found It Difficult To Add Certain Sections

Summary

In general, this assignment was merely a combination of what I started learning when the internship began, with the boiler template and how to start an HTML project to adding the css and knowing how to code the columns and add a navigation bar to adding image source tags.This put my knowledge of front end programming to the test and made me realize that I needed to start from the very beginning by reviewing my earlier code and how I started those projects using the boiler template.
